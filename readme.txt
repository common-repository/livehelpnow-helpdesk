== Installation ==

1. Upload the entire `livehelpnow-helpdesk` folder to the `/wp-content/plugins/` directory, or install plugin from admin panel
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Authenticate livehelpnow account in plugin settings (you can get an account on (https://www.livehelpnow.net/))
4. Sample Chat should be installed by default and enabled on every page
5. More options are available in plugin's settings and on chat edit page
6. You can enable / disable chat for specific pages on page edit screen (side meta boxes)
